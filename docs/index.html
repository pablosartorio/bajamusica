Features

Todo lo que necesitás, nada que no necesitás

Sin cuentas, sin suscripciones, sin extensiones de navegador. Solo buscás, elegís y bajás.

Búsqueda o URL directa
Escribís el nombre del tema o pegás la URL de un video o playlist de YouTube. Los dos caminos funcionan igual.
Playlists completas
Pegás la URL de una playlist y la app la expande entera. Revisás, tildás lo que querés, y descargás.
MP3 o MP4
Elegís el formato antes de bajar. MP3 para audio, MP4 para video. Convertido localmente con ffmpeg.
Progreso en vivo
Panel con barras VU por cada archivo: descargando, convirtiendo, listo. Cerrás el panel y la descarga sigue igual.
Historial de descargas
Cada descarga queda guardada. Un botón te muestra lo que bajaste con fecha, formato y carpeta destino.
Sin cuenta ni registro
No necesitás cuenta de Google ni ningún login. Corre completamente local en tu máquina.
¿Cómo funciona?

Tres pasos y listo

01
Buscás o pegás una URL
Escribís el nombre de un artista, canción o álbum, o pegás directo la URL de un video o playlist de YouTube. El buscador devuelve hasta 30 resultados con miniatura, canal y duración.
02
Elegís qué bajar
Tildás los tracks que querés. Elegís el formato (MP3 / MP4), la calidad, el esquema de nombre y la carpeta de destino en tu máquina. Podés abrir el explorador nativo para elegir la carpeta.
03
Descargás y seguís el progreso
Hacés click en Descargar. Un panel muestra el progreso de cada archivo con barras VU. Cuando termina, los archivos están en tu carpeta, listos para escuchar offline.
Plataformas

Windows y Linux

Mismo código, misma experiencia. En Windows no necesitás instalar nada; en Linux son dos comandos de terminal.

Windows
Zero install
  • Descargás el .exe desde la página de releases
  • Lo ejecutás — se abre el navegador en localhost
  • Python, ffmpeg y dependencias van incluidos en el ejecutable
  • Compatible con Windows 10 y 11
Linux
Python nativo
  • Instalás ffmpeg y python3-tk con apt
  • Clonás el repo y corrés ./run.sh
  • El script crea el entorno virtual e instala las dependencias
  • Probado en Ubuntu 22.04+ y Debian
# Linux — tres comandos y ya
$ sudo apt install ffmpeg python3-tk
$ git clone https://github.com/pablosartorio/bajamusica.git
$ cd bajamusica && ./run.sh

Empezá a bajar ahora

Gratis, sin cuenta, sin límites. El código es abierto y podés ver exactamente qué hace.
